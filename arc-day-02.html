Step 2: Check Your USDC Balance

Create the Balance Checker Script

  1. In VS Code, right-click your arc-project folder and select New File
  2. Name it: check-balance.ts
  3. Copy and paste this code:
import { AppKit } from "@circle-fin/app-kit";
import { createViemAdapterFromPrivateKey } from "@circle-fin/adapter-viem-v2";

// Initialize App Kit
const kit = new AppKit();

const checkBalance = async () => {
  console.log("Checking your USDC balance on Arc Testnet\n");

  // Create adapter from your private key
  const adapter = createViemAdapterFromPrivateKey({
    privateKey: process.env.PRIVATE_KEY as string,
  });

  // Get your wallet address
  const address = await adapter.getAddress();
  console.log("Your wallet address:", address);
  console.log("----------------------------------\n");

  try {
    // Fetch USDC balance on Arc Testnet
    const balance = await kit.getBalance({
      adapter,
      chain: "Arc_Testnet",
      token: "USDC",
    });

    console.log("Balance fetched successfully");
    console.log("----------------------------------");
    console.log("USDC Balance:", balance, "USDC");
    console.log("----------------------------------\n");

  } catch (error: any) {
    console.error("Error fetching balance:", error.message);
    console.log("\nTroubleshooting:");
    console.log("- Check your internet connection");
    console.log("- Verify your private key in .env file");
    console.log("- Make sure you are using Arc Testnet");
  }
};

// Run the balance checker
checkBalance();

Step 3: Check Balance on Multiple Chains

Why This Matters

Arc supports Unified Balance. Your USDC across multiple chains shows as one balance.

Let's see how to check USDC on different chains.

Create Multi-Chain Balance Checker

  1. Right-click your arc-project folder and select New File
  2. Name it: multi-chain-balance.ts
  3. Copy and paste this code:
import { AppKit } from "@circle-fin/app-kit";
import { createViemAdapterFromPrivateKey } from "@circle-fin/adapter-viem-v2";

const kit = new AppKit();

const checkMultiChainBalance = async () => {
  console.log("Checking USDC balance across multiple chains\n");

  const adapter = createViemAdapterFromPrivateKey({
    privateKey: process.env.PRIVATE_KEY as string,
  });

  const address = await adapter.getAddress();
  console.log("Wallet:", address);
  console.log("----------------------------------\n");

  // List of chains to check
  const chains = [
    "Arc_Testnet",
    "Ethereum_Sepolia",
    "Base_Sepolia",
  ];

  let totalBalance = 0;

  for (const chain of chains) {
    try {
      const balance = await kit.getBalance({
        adapter,
        chain: chain as any,
        token: "USDC",
      });

      console.log(`${chain}: ${balance} USDC`);
      totalBalance += parseFloat(balance);
    } catch (error) {
      console.log(`${chain}: Unable to fetch (might not have USDC here)`);
    }
  }

  console.log("\n----------------------------------");
  console.log("Total USDC across all chains:", totalBalance.toFixed(2), "USDC");
  console.log("----------------------------------");
  
  console.log("\nThis is how Arc's Unified Balance works.");
  console.log("You can spend USDC from ANY chain in one transaction.");
};

checkMultiChainBalance();

Step 4: View Transaction History

Create Transaction History Viewer

  1. Right-click your arc-project folder and select New File
  2. Name it: transaction-history.ts
  3. Copy and paste this code:
import { createPublicClient, http, formatUnits } from "viem";
import { defineChain } from "viem";

// Define Arc Testnet chain
const arcTestnet = defineChain({
  id: 5042002,
  name: "Arc Testnet",
  network: "arc-testnet",
  nativeCurrency: { 
    name: "USDC", 
    symbol: "USDC", 
    decimals: 6 
  },
  rpcUrls: {
    default: { http: ["https://rpc.testnet.arc.network"] },
    public: { http: ["https://rpc.testnet.arc.network"] },
  },
  blockExplorers: {
    default: {
      name: "ArcScan",
      url: "https://testnet.arcscan.app",
    },
  },
});

// Create a client to read blockchain data
const client = createPublicClient({
  chain: arcTestnet,
  transport: http(),
});

const getTransactionHistory = async (walletAddress: string) => {
  console.log("Fetching recent transactions on Arc Testnet\n");
  console.log("Wallet:", walletAddress);
  console.log("----------------------------------\n");

  try {
    // Get the latest block number
    const latestBlock = await client.getBlockNumber();
    console.log("Latest block:", latestBlock.toString());
    console.log("Scanning last 20 blocks for transactions\n");

    const transactions: any[] = [];

    // Scan the last 20 blocks
    for (let i = 0; i < 20; i++) {
      const blockNumber = latestBlock - BigInt(i);
      
      try {
        const block = await client.getBlock({ 
          blockNumber,
          includeTransactions: true,
        });

        // Filter transactions involving our wallet
        if (block.transactions) {
          for (const tx of block.transactions as any[]) {
            if (
              tx.from?.toLowerCase() === walletAddress.toLowerCase() ||
              tx.to?.toLowerCase() === walletAddress.toLowerCase()
            ) {
              transactions.push({
                hash: tx.hash,
                from: tx.from,
                to: tx.to,
                value: tx.value,
                blockNumber: block.number,
                timestamp: block.timestamp,
              });
            }
          }
        }
      } catch (error) {
        // Skip blocks that cannot be fetched
        continue;
      }
    }

    // Display results
    if (transactions.length === 0) {
      console.log("No transactions found in the last 20 blocks.");
      console.log("\nTry:");
      console.log("- Sending a transaction (run Day 1 script)");
      console.log("- Checking a different wallet address");
    } else {
      console.log(`Found ${transactions.length} transaction(s):\n`);
      console.log("----------------------------------");

      transactions.forEach((tx, index) => {
        const date = new Date(Number(tx.timestamp) * 1000);
        const direction = tx.from.toLowerCase() === walletAddress.toLowerCase() 
          ? "SENT" 
          : "RECEIVED";
        
        console.log(`\nTransaction #${index + 1}:`);
        console.log(`${direction}`);
        console.log(`From: ${tx.from}`);
        console.log(`To: ${tx.to}`);
        console.log(`Amount: ${formatUnits(tx.value, 6)} USDC`);
        console.log(`Time: ${date.toLocaleString()}`);
        console.log(`Block: ${tx.blockNumber.toString()}`);
        console.log(`Hash: ${tx.hash}`);
        console.log(`View: https://testnet.arcscan.app/tx/${tx.hash}`);
      });

      console.log("\n----------------------------------");
    }

  } catch (error: any) {
    console.error("Error:", error.message);
    console.log("\nMake sure:");
    console.log("- You have internet connection");
    console.log("- The wallet address is valid (starts with 0x)");
  }
};

// Replace with your wallet address
const MY_WALLET = "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb1";

getTransactionHistory(MY_WALLET);

Common Questions

Q1: Why do I need code for this? Rabby already shows my balance.

You are building an APP. Users will use YOUR app, not Rabby. Think about Cash App. Do they tell users to "check Rabby"? No. They fetch the balance and show it in their own beautiful UI.

Q2: What is the difference between balance and transaction history?

Balance = How much you have RIGHT NOW (one number: 18.99 USDC)

History = List of all past transactions (sent $10 to John, received $5 from Sarah, etc.)

Q3: Why scan only 20 blocks? What if my transaction is older?

20 blocks is just for learning. In a real app, you would:

  • Store transactions in a database
  • Use blockchain indexers (like The Graph)
  • Or scan more blocks (but it takes longer)

Q4: Can I check someone else's balance?

YES. Blockchain is public. You can check ANY wallet's balance if you know their address. But you cannot SEND their money. You need their private key for that.

Q5: What is Unified Balance?

Imagine you have:

  • 10 USDC on Ethereum
  • 5 USDC on Polygon
  • 3 USDC on Arc

Arc's Unified Balance shows: 18 USDC total and lets you spend from all chains in one transaction.

Q6: Why does transaction history take longer to load than balance?

Balance is ONE number (fast to fetch). History requires scanning multiple blocks and filtering transactions (slower). Real apps use databases to cache this.

Troubleshooting

Error: Failed to fetch balance

Fix:

  • Check internet connection
  • Verify private key in .env file
  • Make sure you are using Arc_Testnet (not mainnet)

Error: Cannot read properties of undefined

Fix:

  • Make sure you ran npm install in Day 1
  • Restart terminal and try again

Transaction history shows nothing

This is normal if:

  • You have not sent any transactions yet
  • Your transactions are older than 20 blocks

Fix: Run the Day 1 transfer script to create a new transaction.

Balance shows 0.00 USDC

Fix:

  • Get more testnet USDC from https://faucet.circle.com
  • Make sure you are checking the correct wallet address
