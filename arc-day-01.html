Day 1: Wallet Setup + First USDC Transfer on Arc

What You Will Learn Today

  • Connect your wallet to Arc Testnet
  • Get free testnet USDC
  • Send your first USDC transaction using code

Why Arc? Why Not Ethereum or Polygon?

Feature Ethereum Arc
Tokens needed ETH (gas) + USDC (payment) = 2 tokens Only USDC (used for both payment and gas)
Gas cost $5 to $10, changes every minute $0.01, fixed
Confirmation time 3+ minutes Under 1 second
Example:

Your friend has 50 USDC on Ethereum but zero ETH.
Can they send you 10 USDC? NO -> they need ETH for gas.

Your friend has 50 USDC on Arc.
Can they send you 10 USDC? YES -> USDC pays for gas.

Prerequisites

1. VS Code

  • Download and install VS Code on your PC.
  • Create a folder anywhere on your drive.
  • In VS Code, click File (top bar) -> Open Folder -> Choose your folder.

2. Node.js

  • Download and install Node.js on your PC.
  • To verify, open VS Code terminal and type:
node -v
npm -v

If any error shows in your PowerShell then at terminal paste this:

Switch VS Code Terminal from PowerShell to CMD

Temporarily:

  1. Open VS Code -> Terminal -> New Terminal (Ctrl + Shift + ~)
  2. Click the dropdown next to + -> select Command Prompt

Permanently (set as default terminal):

  1. Press Ctrl + , to open Settings
  2. Search: terminal.integrated.defaultProfile.windows
  3. Set it to: "Command Prompt"

Allow PowerShell Scripts to Run (Optional)

If you still need PowerShell scripts to run:

  1. Open PowerShell as Administrator
  2. Run this command:
Set-ExecutionPolicy RemoteSigned
  1. Press Y to confirm

This allows local scripts to run safely.

Revert PowerShell Execution Policy (Do this when your work is finished)

Note: Do this only when your work is finished, otherwise scripts will stop working again.

  1. Open PowerShell as Administrator
  2. Check current policy first (optional):
Get-ExecutionPolicy -List
  1. Revert back to the default restricted policy:
Set-ExecutionPolicy Restricted -Scope LocalMachine
  1. Confirm with Y

Scripts will now be blocked again like before.

You can always revert back after your work is done, or change the terminal to CMD instead of PowerShell.

Step 1: Install Rabby Wallet

  1. Go to https://www.rabby.io
  2. Click Download
  3. Select your browser (Chrome, Firefox, etc.)
  4. Click Add to [Your Browser]
  5. Click the Rabby icon in your browser toolbar
  6. Click Create a new wallet
  7. Create a password
  8. Backup your seed phrase
  9. Confirm your seed phrase by clicking the words in order
  10. Your wallet is ready

Step 2: Connect Rabby to Arc Testnet

  1. Open Rabby
  2. Click the settings icon at the top
  3. Click Custom Network
  4. Fill in these exact details:
Network name: Arc Testnet
RPC URL:      https://rpc.testnet.arc.network
Chain ID:     5042002
Currency:     USDC
Explorer:     https://testnet.arcscan.app
  1. Click Save
  2. Switch to Arc Testnet from the dropdown

Step 3: Get Free Testnet USDC

On Arc, USDC is used for both sending money and paying transaction fees (gas).

  1. Go to https://faucet.circle.com
  2. Select "Arc Testnet" from the network options
  3. Add your wallet address
  4. Click Send 20 USDC
  5. You should now see 20 USDC in your wallet

Step 4: Create Your First Arc Project

Open Terminal in VS Code

  1. Open VS Code
  2. Click Terminal -> New Terminal (top bar)
  3. You will see a command prompt at the bottom

Create Project Folder

Copy and paste these commands one at a time:

mkdir arc-day-01
cd arc-day-01
npm init -y

This creates a folder called arc-day-01 and a package.json file that tracks your project info.

Install Required Libraries

npm install @circle-fin/app-kit @circle-fin/adapter-viem-v2 viem typescript tsx

What was installed:

  • @circle-fin/app-kit -> Circle's tool for sending USDC
  • viem -> Library for talking to blockchains
  • typescript -> Better JavaScript with types
  • tsx -> Runs TypeScript code

This takes 1 to 2 minutes. Wait for it to finish.

Step 5: Get Your Private Key from Rabby

  1. Open Rabby
  2. Click the settings icon (top right)
  3. Click Manage Address -> Click on Address
  4. Click Backup Private Key
  5. Enter your Rabby password
  6. Click Confirm
  7. Copy the private key (it looks like: 0x1234abcd...)

Step 6: Create the .env File

  1. In VS Code, right-click your project folder and select New File
  2. Name the file exactly: .env
  3. Paste the following inside:
PRIVATE_KEY=your_wallet_private_key_here
  1. Replace your_wallet_private_key_here with the private key you copied
  2. Save the file (Ctrl+S)

Example:

PRIVATE_KEY=0x12345.....

Step 7: Write Your First Transfer Script

  1. Right-click your project folder in VS Code and select New File
  2. Name it: transfer.ts
  3. Copy and paste this exact code:
// AppKit handles blockchain actions
import { AppKit } from "@circle-fin/app-kit";
// Viem adapter signs transactions with your private key
import { createViemAdapterFromPrivateKey } from "@circle-fin/adapter-viem-v2";
// SendParams ensures safe, typed transaction data
import type { SendParams } from "@circle-fin/app-kit";

// Create an instance of App Kit
const kit = new AppKit();

const sendUSDC = async () => {
  console.log("Starting USDC transfer on Arc Testnet\n");

  // Step 1: Create adapter from your private key
  const adapter = createViemAdapterFromPrivateKey({
    privateKey: process.env.PRIVATE_KEY as string,
  });

  console.log("Wallet connected");

  // Step 2: Define the transfer details
  const sendParams: SendParams = {
    from: {
      adapter,
      chain: "Arc_Testnet"
    },
    to: "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb1", // Replace with your recipient
    amount: "1.00", // Sending 1 USDC
    token: "USDC",
  };

  console.log("Preparing to send:", sendParams.amount, "USDC");
  console.log("To address:", sendParams.to);

  // Step 3: Estimate the gas fee first
  console.log("\nEstimating transaction cost...");
  const estimate = await kit.estimateSend(sendParams);
  console.log("Estimated gas fee:", estimate.fee, "USDC");
  console.log(
    "Total cost:",
    (parseFloat(sendParams.amount) + parseFloat(estimate.fee)).toFixed(6),
    "USDC"
  );

  // Step 4: Send the transaction
  console.log("\nSending transaction...");
  const result = await kit.send(sendParams);

  // Step 5: Show results
  console.log("\nSUCCESS! Transaction completed");
  console.log("----------------------------------");
  console.log("Transaction Hash:", result.txHash);
  console.log("View on Explorer:", result.explorerUrl);
  console.log("----------------------------------");
};

// Run the transfer function
sendUSDC().catch((error) => {
  console.error("Error:", error.message);
  process.exit(1);
});
  1. Save the file (Ctrl+S)

In the VS Code terminal, run:

Step 9: Verify Your Transaction

View on Block Explorer

  1. Copy the explorer URL from your terminal output
  2. Paste it in your browser
  3. You will see:
    • Status: Success
    • From: Your wallet address
    • To: Recipient address
    • Value: 1.00 USDC
    • Gas Fee: ~0.01 USDC
    • Confirmation time: under 1 second

Check Your Rabby Balance

  1. Open Rabby
  2. Your balance should now reflect:
    • 1.00 USDC sent
    • 0.01 USDC used as gas fee
    • Remaining balance updated accordingly

Troubleshooting

Error: Cannot find module '@circle-fin/app-kit'

Fix: Run the following command:

npm install @circle-fin/app-kit @circle-fin/adapter-viem-v2 viem

Error: Private key is not defined

Fix: Check your .env file. Make sure it contains:

PRIVATE_KEY=0x...

Error: Insufficient funds

Fix: Get more testnet USDC from https://faucet.circle.com

Error: Invalid address

Fix: Make sure the recipient address starts with 0x and is exactly 42 characters long.
